📖 Visualization Terminology

🔗 Link Distance

Controls spacing between directly connected entities within the same cluster. Lower values create tighter clusters, higher values spread entities apart.

☁️ Cloud Separation

Controls the distance between separate entity clusters/clouds. Higher values push cluster clouds further apart across the visualization space.

🎯 Perspective View (Radial Layout)

When selecting a category (e.g., People), those entities are pulled to the center forming a hub, while other categories radiate outward in a spoke pattern. Creates a focused view showing how the selected category connects to everything else.

🏷️ Cluster Anchoring

Each cluster is named after its highest-confidence entity (the "anchor"), providing specific identification rather than generic category names.

⚡ Force Simulation

Uses physics-based forces (attraction, repulsion, collision) to automatically position entities. Link forces pull connected entities together, charge forces push all entities apart, creating natural clustering.

🔄 Live Controls

Real-time sliders that immediately adjust the visualization without requiring a refresh. Allows for interactive exploration and fine-tuning of the network layout.
